带有Nios II软核的FPGA项目Jic文件打包方法

2017-06-29
FPGA
FPGA, Nios II
  1. 将.sof文件和.elf文件复制到同一文件夹下，如d:/bin

  2. 编辑nios脚本程序 build.sh

  3. 生成Hex文件
    运行Nios II Command Shell
    跳转至上述文路径，如 d:/bin

    运行 build.sh在当前文件夹下生成.hex文件


  4. 将上述命令生成的sof_elf.hex转换成.jic文件

    • 在Quartus®软件中，open File > Convert Programming Files > Set the programming file as JTAG Indirect Configuration File (.jic).
    • 在配置下拉菜单中选择合适大小的EPCS器件
    • 命名你的输出.jic 文件
    • 点击Flash Loader的下面，在右边选择Add Device，从列表中选择你的FPGA器件
    • 选中Sof Page , Remove
    • 点击Add Hex data，选择Relative addressing，选择上面生成的.hex 文件
    • 点击Generate可看到目标路径下生成了.jic文件。
